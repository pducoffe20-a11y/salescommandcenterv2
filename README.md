# Sales Command Center V2

A polished Vite + React + TypeScript sales command center for a senior account
executive. The app is an action-first revenue cockpit with account workspaces,
meeting prep, printable briefs, follow-up drafts, voice review, and prospect
scoring.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

The production build outputs to `dist/`, which Vercel can deploy from the
standard Vite defaults.
